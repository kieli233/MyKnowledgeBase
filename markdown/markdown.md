# markdown使用指南  

## 基础  

```#```  
一级标题，每加一个下一级标题最多加到6个，这个对应的是HTML中的```<H1>```标签  

```" ` ` `code` ` ` "```  
一般用连续三个" ` "来代表代码块  

```---```  
用连续的三个横线表示划分线同样操作的有" * "和" _ "  

```[名字](#引导位置)```  
作用大概就是目录，能引导到指定位置，如果文章过长可以用这个引导别人到指定位置获取想要的信息

```![名字](图片地址)```  
这条主要是我自己喜欢用表情包才去查怎么用的
